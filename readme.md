# Recycling Production Line Manager Selection System

## Overview
A standalone system to evaluate and rank candidates using database logic, AI prompting, and a React dashboard.

## Tech Stack
- MySQL
- Node.js (Faker.js)
- React + Vite
- Mantine UI

## Setup
1. Import `database/schema.sql`
2. Run Faker generator for sample data
3. Start frontend:
   npm install
   npm run dev

# Run frontend and backend with
1. frontend - cd frontend -> npm run dev
2. to generate sample_candidates - cd generator -> node generatorCandidates.js

## Features
- Auto-ranking using SQL triggers
- AI-based evaluation prompts
- Interactive dashboard with leaderboard and candidate cards
